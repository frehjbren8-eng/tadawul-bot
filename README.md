# 🤖 بوت كاشف اختراق المقاومة — تداول

بوت تيليغرام يكتشف أسهم السوق السعودي التي اخترقت المتوسط المتحرك (MA).

---

## الخطوات (15 دقيقة فقط)

### 1. أنشئ البوت في تيليغرام
- افتح @BotFather في تيليغرام
- أرسل `/newbot`
- احفظ الـ **Token** (مثال: `123456:ABC-DEF...`)

### 2. احصل على Chat ID
- افتح @userinfobot في تيليغرام
- أرسل أي رسالة → سيعطيك الـ **ID**

### 3. ارفع على Railway
1. اذهب إلى [railway.app](https://railway.app) وسجل بحساب GitHub
2. اضغط **New Project → Deploy from GitHub repo**
3. ارفع هذه الملفات على GitHub أولاً، ثم اختر الـ repo
4. في Railway → اضغط على المشروع → **Variables** → أضف:

```
TELEGRAM_TOKEN   = توكن البوت من BotFather
ALPHA_VANTAGE_KEY = TQ9F7FW318K1F0XR
CHAT_ID          = الـ ID بتاعك من userinfobot
```

5. اضغط **Deploy** ✅

## الأوامر
| الأمر | الوظيفة |
|-------|---------|
| `/start` | تشغيل البوت |
| `/scan` | مسح فوري الآن |
| `/status` | حالة البوت |

📅 يرسل تقرير تلقائي **كل يوم الساعة 9 صباحاً** (توقيت الرياض).
